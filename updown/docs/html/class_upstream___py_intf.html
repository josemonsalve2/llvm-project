void Upstream_PyIntf::addSystemPaths ( )
private

Adds system paths for searching the necessary modules.

This function adds the following system locations to the python system path:

  • Installation path for updown runtime lib emulator
  • Source code location for updown runtime lib emulator
  • A folder named emulator in the current location
