Detailed Description

Wrapper class that allows simulation of runtime calls.

This class inherits from UDRuntime_t, and it overwrites the methods such that they can be simulated using a memory region.

This class does not use polymorphism. It just oversubscribes the methods, wrapping the original implementation of the runtime

void UpDown::SimUDRuntime_t::initMemoryArrays ( )

Allocate memory for the simulation of the updown.

This function allocates the MappedMemory, the Scratchpad memory and the Control Memory, used for the simulation. Then it changes the MachineConfig object to use these pointers instead of the default ones for these three memory regions.
