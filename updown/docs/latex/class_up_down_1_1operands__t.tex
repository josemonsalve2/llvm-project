\doxysubsection{Detailed Description}
Class that holds information about the operands. 

Contains a pointer to data and the number of operands. data is an array of ptr\+\_\+t, representing each word in the operand buffer when copied.

Num\+Operands is 8 bits long. Up to 256 operands are allowed
